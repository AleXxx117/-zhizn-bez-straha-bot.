# Готовое оформление в BotFather

Эти тексты можно скопировать в `@BotFather`.

## Название — `/setname`

```text
Жизнь без страха | Поддержка
```

## Коротко о боте — `/setabouttext`

```text
Тёплая поддержка при тревоге, панических волнах и навязчивых мыслях. Маленькими шагами возвращаемся в жизнь.
```

## Описание — `/setdescription`

```text
Я рядом, когда тревожно, страшно выходить из дома, накрывает паническая волна или кажется, что случился откат. Здесь можно выговориться, выбрать короткую практику и получить поддержку в стиле сообщества «Жизнь без страха». Бот не ставит диагнозы и не заменяет врача или экстренную помощь.
```

## Команды — `/setcommands`

```text
start - Начать и открыть меню
support - Получить поддержку
practice - Практика на 5 минут
daily - Ежедневная поддержка
help - Возможности и безопасность
privacy - Какие данные сохраняются
id - Показать ID этого чата
```
